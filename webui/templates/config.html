服务 端口 协议 说明

⚠️ 防火墙设置

请确保以下端口在防火墙中开放，否则客户端无法正常 PXE 启动：

iptables -A INPUT -p udp --dport 67 -j ACCEPT
iptables -A INPUT -p udp --dport 69 -j ACCEPT
iptables -A INPUT -p tcp --dport 2049 -j ACCEPT
iptables -A INPUT -p tcp --dport 20048 -j ACCEPT

📝 DHCP 配置预览

# DHCP Server Configuration
# Server IP: {{ config.server_ip }}
# Subnet: {{ config.subnet_mask }}

authoritative;
ddns-update-style none;

subnet {{ config.server_ip.rsplit('.', 1)[0] }}.0 netmask {{ config.subnet_mask }} {
    range {{ config.dhcp_range_start }} {{ config.dhcp_range_end }};
    option routers {{ config.server_ip }};
    option subnet-mask {{ config.subnet_mask }};
    option domain-name-servers {{ config.server_ip }};
    
    # PXE Boot Options
    filename "pxelinux.0";
    next-server {{ config.server_ip }};
}

📂 NFS 导出配置

# /etc/exports
# NFS Share for PXE Clients

/nfsroot    *(ro,sync,no_subtree_check,no_root_squash,async,insecure)

客户端将通过 NFS 挂载 ISO 内容，避免下载到内存

📁 数据目录

目录 用途
/app/data/iso 存放上传的 ISO 文件
/app/data/boot 提取的 PXE boot 文件
/app/data/nfs NFS 共享目录，存放 ISO 内容
/app/data/temp 临时文件目录
